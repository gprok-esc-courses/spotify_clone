# Spotify clone

This is a student project.

## How to clone
1. Clone the project
2. Setup venv in PyCharm
3. ```python manage.py migrate```
4. ``` python manage.py createsuperuser```
5. Create a ```media``` folder with ```covers``` and ```songs``` subfolders
